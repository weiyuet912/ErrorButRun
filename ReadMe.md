# Lang Sushi - Online Restaurant Ordering System

**A Full-Stack digital solution for traditional Japanese dining, following the "Old is Gold" theme.**

`Lang Sushi` is a comprehensive web-based ordering platform designed to streamline restaurant operations. This project marks our transition from simple scripted logic to a robust **Flask + SQLite** architecture, ensuring data persistence, security, and a seamless user experience.



## Key Features

### For Customers

- **Dynamic Menu Interface**: A responsive menu where users can browse dishes with real-time price and quantity updates.
- **Instant Cart Logic**: Add or remove items with immediate visual feedback—no page reloads required.
- **Secure Checkout**: Integration of user sessions to link customer identities with their specific orders securely.

### For Restaurant Staff

- **Protected Admin Portal**: A dedicated dashboard secured by authentication for viewing incoming orders.
- **Live Order Tracking**: Orders are displayed in reverse chronological order, allowing chefs to prioritize the latest requests.
- **Automated Invoicing**: Every order is assigned a unique, sequential ID (e.g., `#001`) generated by the backend.



### Tech Stack

- **Frontend**: HTML5, CSS3 (featuring Glassmorphism UI), JavaScript (ES6+ Fetch API).
- **Backend**: Python 3.x, Flask Web Framework.
- **Database**: SQLite with Flask-SQLAlchemy (ORM).
- **State Management**: Encrypted Server-side Sessions.



## Installation & Setup

### 1. Clone the repository

Bash

```
git clone https://github.com/your-username/lang-sushi.git
cd lang-sushi
```

### 2. Install dependencies

Bash

```
pip install flask flask-sqlalchemy
```

### 3. Run the application

The system features **Auto-initialization**. It will automatically detect and build the SQLite database schema on its first run:

Bash

```
python app.py
```

Open your browser and navigate to `http://127.0.0.1:5000`.
